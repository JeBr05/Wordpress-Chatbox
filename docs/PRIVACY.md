# Privacy

This plugin sends visitor questions to OpenAI through your WordPress server.

When knowledge base sync is used, selected WordPress pages and posts are uploaded to OpenAI as files and attached to a vector store.

The API key is stored in WordPress options in encrypted form using WordPress salts.

The plugin can store conversations for analytics. You can disable this in Security. When logging is enabled, IP addresses are hashed and email addresses and phone numbers can be redacted before storage.

Suggested privacy policy text:

This website uses an AI chatbot to answer questions. When you use the chatbot, your message is processed by our website and sent to our AI provider to generate an answer. We may store chatbot messages to improve service quality. Do not share sensitive personal information in the chatbot.
